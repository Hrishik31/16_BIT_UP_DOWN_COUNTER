# 16_BIT_UP_DOWN_COUNTER
 16-bit Up/Down Counter – RTL to GDSII ASIC Flow  This repository documents the complete ASIC design flow for a 16-bit synchronous Up/Down Counter implemented in Verilog HDL. The project was completed as part of a hands-on VLSI workshop using Synopsys tools at Pandit Deendayal Energy University (PDEU) in collaboration with ChipIN Centre.
